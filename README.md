# `ttv-live`

A simple command line program to check whether your favourite Twitch streamers are currently live. 

![screenshot](https://user-images.githubusercontent.com/16789070/182046526-6cc16a6d-32e1-4902-aef9-af8a7905b72b.png)

## Installation

Pre-built binaries are available from the [releases page](https://github.com/chowder/ttv-live/releases).

## Configuration 

Populate your `~/.config/ttv-live/ttv-live.config` file with the list of streamers to track.

```json
{
  "streamers": [
    "xqc",
    "skillspecs",
    "dismellion",
    "dino_xx",
    "usteppin"
  ]
}
```
